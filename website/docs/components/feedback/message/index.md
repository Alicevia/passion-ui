## 显示时间控制

::: danger
  `duration` 呈现时间 是从出现动画开始后开始计时
:::
::: tip
`closable` 可以控制是否显示关闭按钮 
`forceClose` 强制关闭默认是关闭的 这就意味着只要鼠标在消息上那么不管结束时间到没到都不会消失
:::
::: demo 
feedback/message/duration
:::
